# Documentation

This folder contains the public project documentation for `whpc-kit`.

## Start Here

1. [Getting Started](GETTING_STARTED.md)
2. [Reproducibility Notes](REPRODUCIBILITY_NOTES.md)
3. [Project Scope](SCOPE.md)
4. [Product Roadmap](PRODUCT_ROADMAP.md)
5. [API Naming Guide](API_NAMING.md)

Most visitors only need the first three documents. The roadmap and naming guide are mainly useful for contributors and maintainers.

## What Each File Covers

1. `GETTING_STARTED.md`: installation, smoke checks, and first commands.
2. `REPRODUCIBILITY_NOTES.md`: public reproducibility contract and the native versus companion-checkout split.
3. `SCOPE.md`: boundaries of the repository and product direction.
4. `PRODUCT_ROADMAP.md`: staged evolution of the package and possible post-v1.0 layers.
5. `API_NAMING.md`: target public naming for estimators and helpers.
